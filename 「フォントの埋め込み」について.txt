# 「フォントの埋め込み」について by 旭 2020/10/13


## この文章の目的

日本語で TeX から PDF を作成する際につまづきやすい
日本語フォントの埋め込みについて，説明した文章です．
TeXLive 2020 を Windows 10 で使用することを想定しています．
他の TeX ディストリビューションやOS環境については，自分で調査してください．


## まず何をすればよいですか？

以下の公式文章を読め．

日本語 TEX 開発コミュニティ : kanji-config-updmap — 日本語フォント設定ツール
http://ctan.math.washington.edu/tex-archive/fonts/ptex-fontmaps/kanji-config-updmap.pdf

以上．


## もう少し具体的に教えて

多分，以下の手順を踏むと思われます．
なお，以下を実行して，PCの環境が壊れても当方は関知しないため，
*自己責任*でお願いします．
ちなみに TeXLive 2020 では，デフォルトで
「原ノ味フォント」が埋め込まれるみたいなので，特別な設定は不要です．
（どのフォントが埋め込まれるか，念のため確認は必要と思います．）

* Windows ユーザーの場合
  - [スタート] → [TeX Live 2020] → [TeX Live command-line]
    とたどり，TeX Live のコマンドプロンプトを開きます．
  - kanji-config-updmap-sys status を実行します．
    CURRENT family for ja: のところに出てきたフォントが埋め込まれます．
    また，Standby family : のところに出てきたフォントは，変更可能な一覧です．
    * kanji-config-updmap-sys を実行するとき，カレントディレクトリのパスに
      日本語文字が含まれると実行に失敗します．デフォルトではカレントディレクトリが
      ユーザーフォルダとなるため，ユーザー名を日本語にしている人は，気を付けてください！
  - 変更するには，kanji-config-updmap-sys <fontname> とします．
    ここで， <fontname> は，前項の Standby family から指定します．
    例： kanji-config-updmap-sys ms
  - ちなみに ms を埋め込むと Word 文章と似た見た目になると思います．
  - 公式文書によると yu-win10 を埋め込むと，一部記号がおかしくなるみたいです．
    したがって，新しい Word の游フォントに見た目を近づけるのはお勧めしません．
  - 公式文書によると ipaex を埋め込むと，細字の明朝や太字のゴシックが利用できないようです．
    他のフォントでもプリセットされているフォントのウェイト（文字の太さ）に注意が必要みたいです．


* Mac や Linux ユーザーの場合
  - 私にはよくわかりませんので，各自で調査してください．以下は推測で書いています．
  - とりあえず，ターミナル（端末）を開きます．
  - $ sudo kanji-config-updmap-sys status を実行して，
    CURRENT family for ja: のところに出てきたフォントが埋め込まれます．
    また，Standby family : のところに出てきたフォントは，変更可能な一覧です．
  - 変更するには，$ sudo kanji-config-updmap-sys <fontname> とします．
    ここで， <fontname> は，前項の Standby family から指定します．
    例： $ sudo kanji-config-updmap-sys ms
  - ちなみに ms を埋め込むと Word 文章と似た見た目になると思います．
  - 公式文書によると yu-win10 を埋め込むと，一部記号がおかしくなるみたいです．
    したがって，新しい Word の游フォントに見た目を近づけるのはお勧めしません．
  - 公式文書によると ipaex を埋め込むと，細字の明朝や太字のゴシックが利用できないようです．
    他のフォントでもプリセットされているフォントのウェイト（文字の太さ）に注意が必要みたいです．
